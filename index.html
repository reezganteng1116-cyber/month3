tugas hari pertama

Membuat website portofolio sederhana menggunakan HTML, CSS, dan JavaScript.

Tiga fremwork dan library JavaScript:

  • React.js
  • Vue.js
  • Angular

React.js: adalah library JavaScript untuk membangun antarmuka pengguna, Vue.js adalah framework progresif untuk membangun UI, dan Angular adalah platform untuk membangun aplikasi web.

  • Dikembangkan oleh Facebook.
  • Fokus pada pembuatan UI (User Interface) berbasis komponen.
  • Banyak digunakan untuk aplikasi web interaktif dan SPA (Single Page Application).

Vue.js: adalah framework progresif untuk membangun antarmuka pengguna, dan Angular adalah platform untuk membangun aplikasi web.

  • Ringan, fleksibel, dan mudah dipelajari.
  • Cocok untuk integrasi bertahap (bisa dipakai sebagian atau penuh).
  • Banyak digunakan untuk membuat UI yang reaktif dengan sintaks yang sederhana.

Angular: adalah platform untuk membangun aplikasi web.

  • Dikembangkan oleh Google.
  • Framework full-featured untuk membangun aplikasi web skala besar.
  • Mendukung MVC/MVVM, data binding dua arah, dependency injection, dan tools bawaan lengkap.
